## 요구사항

### 1. HTML 마크업

- [x] 1-1. 로그인 바 영역의 링크 항목은 로그인, 회원가입, 고객센터, 쿠팡플레이, 제휴마케팅 순서로 마크업해주세요. 이때 5개의 링크 항목은 1개의 목록 요소(ul 요소) 안에 포함되도록 구성해주세요.마크업 시 왼쪽 링크 그룹(쿠팡플레이, 제휴마케팅)과 오른쪽 링크 그룹(로그인, 회원가입, 고객센터)으로 나누어 구성하지 않도록 주의해주세요.
- [x] 1-2. 로고는 대제목의 의미를 가지는 `<h1>` 요소의 콘텐츠로 마크업해주세요.
- [x] 1-3. 로고는 `<h1>` 요소 내 “Coupang” 이라는 텍스트로 마크업해주세요.
- [x] 1-4. 로고를 클릭 시 쿠팡 홈페이지로 이동하도록 하이퍼링크를 지정하고 해당 페이지가 새 창에 보여지도록 구현해 주세요. 이때 하이퍼링크의 보안 취약점을 보완하기 위한 속성도 선언해주세요.
- [x] 1-5. 카테고리 버튼의 경우 `<button>` 요소로 마크업하고 버튼 텍스트는 “카테고리”로 제공해주세요.또한 해당 버튼의 클래스는 button--category로 지정해주세요.(button--category 클래스 이름은 button과 category 사이에 하이픈이 두개, 즉 더블 대시로 구분되어 있음.)
- [x] 1-6. 검색 영역은 `<form>` 관련 요소를 사용하여 마크업해주세요.
- [x] 1-7. 검색분류 영역은 `<select>` 요소로 마크업 하고 DropDown 모양의 아이콘은 `<svg>` 요소로 구현해주세요.
- [x] 1-8. 검색어 입력 서식은 HTML5에 새롭게 추가된 `<input>` 요소의 type 속성 값 중 “검색”의 의미를 가지는 요소로 마크업 해주세요. 이때 해당 입력 서식은 필수 입력 서식임을 알 수 있도록 하고 입력 값에 대한 힌트로 “찾고 싶은 상품을 검색해 보세요!”를 제공해 주세요.
- [x] 1-9. 음성 검색 및 키워드 검색은 `<button>` 요소로 마크업해주세요. 이때 스크린리더 사용자가 해당 버튼의 용도를 이해할 수 있도록 정보를 제공해주세요.
- [x] 1-10. 마이쿠팡은 `<div>` 요소 안에 `<button>` 요소와 링크 목록 영역(`<ul`> 요소)으로 구성하여 마크업하고 마이쿠팡 영역인 `<div>` 요소에는 myCoupang이라는 클래스 이름을 부여해주세요. 이때 `<button>` 요소에는 “마이쿠팡"이라는 텍스트만으로 구성하고 아이콘은 추후 CSS의 배경이미지로 구현해주세요.
- [x] 1-11. 장바구니 영역은 `<div>` 요소로 마크업하고 클래스 이름은 cart로 지정해주세요. 이때 장바구니 영역안에는 장바구니 버튼(`<button`>)과 선택한 상품 개수(0개), 그리고 장바구니가 비어있음을 알리는 메시지 영역으로 구성하여 마크업 해주세요. 이때 장바구니 버튼(`<button`> 요소)에는 “장바구니"라는 텍스트만으로 구성하고 카트 아이콘은 추후 CSS의 배경이미지로 구현해주세요.
- [x] 1-12. 로켓배송 부터 여행티켓까지 링크 영역은 목록으로 구성하고 로켓배송, 로켓프레시, New 아이콘의 경우 `<img>` 요소를 사용하여 마크업해주세요 이때 `<img>` 요소에는 적절한 대체텍스트를 제공해 주세요.
- [x] 1-13. 문법 오류가 발생하지 않도록 작성해주세요. HTML 문법 검사 URL : https://validator.w3.org/

- [x] 2-2. 로그인바 영역의 경우 링크 목록은 화면 가운데 배치하고 `display: flex`를 사용하여 가로로 정렬해주세요. 이때 쿠팡플레이와 제휴마케팅 링크 목록은 왼쪽에 로그인, 회원가입, 고객센터 링크 목록은 오른쪽에 배치되도록 `margin 속성`을 활용하여 구현해주세요.
