# 미션2.오늘의 쇼핑제안

## 주요내용

#### `aria-label`, `role`를 사용해 스크린리더 사용자에게 상품정보 제공하기

- 상품 정보는 `span` 태그에 담겨있어 `role`, `aria-label` 속성을 부여했음
- `img` 태그의 `alt` 속성 등 중복 정보는 `aria-hidden="true'`를 사용해 중복을 피함
- 크롬 익스텐션 스크린리더를 사용해 상품정보가 전부 음성으로 확인되는 것 확인
