# 미션1.오늘의 발견 작성완료

## 주요내용

#### 웹폰트 선언 방식

- 웹폰트 선언(1-5, 2-1)은 html 파일에 `link` 태그로 import하는 방법, css 파일에 `@import`로 넣는 방법 중 전자를 사용
- 두 방식의 차이를 찾아보니, `@import` 방식은 직렬방식으로 다운로드하기 때문에 전체 로딩시간이 길어지고, 스타일 시트 내에서 병합이 일어나서 스타일 시트를 파악하기 어려워지기 때문에 사용하지 않음

#### 구매하기 버튼(화살표 버튼) word-break 설정

- 화살표 버튼이 가로로 길어지면서 '구매하기' 텍스트가 삽입이 되는데 삽입되는 순간에는 텍스트 길이보다 버튼 길이가 짧아서 텍스트가 줄바꿈되어 깨지는 문제가 발생
- 버튼의 `word-break` 설정을 `keep-all`로 적용해서 텍스트가 줄바꿈되지 않게 함
